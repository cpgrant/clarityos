# Operator Playbooks

These playbooks describe the current `v1.0` trusted-runtime maintenance flows.

Use them with:

- `GET /operator/profile`
- `GET /queue/health`
- `GET /workers/health`
- `GET /incidents/workflows/{workflow_id}`
- `GET /incidents/workflows/{workflow_id}/summary`

Available playbooks:

- `migration.md`
- `queue-cleanup.md`
- `worker-repair.md`
- `workflow-recovery.md`
- `incident-response.md`
