# Operator Playbooks

These playbooks describe the current released runtime and operator-facing maintenance flows.

Use them with:

- `GET /operator/profile`
- `GET /queue/health`
- `GET /workers/health`
- `GET /incidents/workflows/{workflow_id}`
- `GET /incidents/workflows/{workflow_id}/summary`

Available playbooks:

- `assistant-surface-incidents.md`
- `session-cleanup.md`
- `widget-deployment.md`
- `migration.md`
- `queue-cleanup.md`
- `operator-runtime.md`
- `storage-backup.md`
- `worker-repair.md`
- `workflow-recovery.md`
- `incident-response.md`
